'සිංහල ගීතිකා එකතුව (Sinhala Hymns & Worship Collection)'

විස්තරය (Description): මෙම ගීතිකා එකතුව දෙවියන් වහන්සේගේ මහිමය උදෙසාත්, කිතුනු බැතිමතුන්ගේ ආධ්‍යාත්මික වර්ධනය උදෙසාත් සකස් කරන ලද ලාභ නොලබන (Non-profit) සේවාවකි. මෙහි අරමුණ විසිරී ඇති සිංහල ගීතිකා එකම තැනකට ගොනු කර, ඕනෑම අයෙකුට පහසුවෙන් ඇසීමට සැලැස්වීමයි.

-හිමිකම් ප්‍රකාශය (Copyright Notice):

* මෙම ගීතිකා වල සම්පූර්ණ අයිතිය සහ ගෞරවය ඒවායේ මුල් නිර්මාණකරුවන්ට (ගී පද රචකයන්, සංගීතවේදීන් සහ ගායන ශිල්පීන්ට) හිමි වේ.

* මෙය හුදෙක් රසවිඳීම සහ නමස්කාර කටයුතු සඳහා පමණක් වන අතර, කිසිදු වාණිජමය හෝ ලාභ ලැබීමේ අරමුණකින් තොරව සිදු කරන්නකි.

* ඔබ යම් නිර්මාණයක හිමිකරුවෙකු නම් සහ මෙය මෙහි ප්‍රදර්ශනය කිරීම පිළිබඳව අකමැත්තක් දක්වන්නේ නම්, කරුණාකර මා අමතන්න. මම වහාම එය ඉවත් කිරීමට කටයුතු කරන්නෙමි.
